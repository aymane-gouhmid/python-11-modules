# Module 00 - Python Fundamentals

## Overview
This module introduces the fundamentals of Python programming through simple garden-themed exercises.

## Concepts Learned

### 1. Functions
- Creating Python functions using `def`
- Calling functions
- Returning and printing values

### 2. Input and Output
- Using `input()`
- Using `print()`

### 3. Variables and Data Types
- Integer values
- String values
- Variable assignment

### 4. Arithmetic Operations
- Addition
- Multiplication
- Calculating totals and areas

### 5. Conditions
- Using `if` and `else`
- Comparing values

### 6. Loops and Recursion
- Using `range()`
- Iteration with loops
- Recursive functions

### 7. Type Annotations
- Function parameter types
- Return type annotations

## Exercises
- Hello Garden
- Garden Plot Area
- Harvest Total
- Plant Age Check
- Water Reminder
- Count to Harvest
- Garden Summary
- Seed Inventory

## Skills Gained
- Writing clean Python functions.
